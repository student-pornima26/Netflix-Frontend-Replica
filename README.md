# 🎬 Netflix Frontend Replica (HTML & CSS)

## 📌 Project Overview

This is a Netflix Clone built using pure HTML and CSS.  
The project replicates the Netflix landing page design including hero banner, navigation bar, movie sections, and footer.

The main goal of this project was to improve my frontend development skills and practice real-world UI design using HTML and CSS.

---

## 🚀 Features

- Netflix-style landing page
- Hero banner section with background image
- Navigation bar with logo and sign-in button
- Movie/TV show rows layout
- Hover effects on movie thumbnails
- Footer section
- Responsive design 

---

## 🛠️ Technologies Used

- HTML
- CSS
- Flexbox
- CSS Grid 
- Media Queries (for responsiveness)


## 💡 What I Learned

- Creating modern UI layouts
- CSS positioning and layering
- Using Flexbox & Grid for alignment
- Working with background images
- Implementing hover effects
- Improving design consistency
